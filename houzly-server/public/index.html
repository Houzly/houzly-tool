Import da Smoobu
Come Importare

Smoobu non permette connessioni dirette browser-to-API. Hai 3 opzioni:

Opzione 1 — CSV Smoobu (più veloce)
Da Smoobu → Reports → Reservations → Export CSV. Trascina qui sotto.
Opzione 2 — API Key (richiede server)
Smoobu → Account → API → copia la chiave. Serve un piccolo proxy Node.js/PHP.
Opzione 3 — Zapier / Make (automatica)
Collega Smoobu a Google Sheet via Zapier, esporta CSV e importa qui.
